# etenbijdaan.nl

Website voor **Eten bij Daan** — gestart als getrouwe kopie van het
voormalige Restaurant Voldaan (Nijmegen-Oost). Statische site op
Cloudflare Workers Static Assets.

```bash
export PATH="/opt/homebrew/opt/node@22/bin:$PATH"
npm install
npm run dev      # lokale preview
npm run deploy   # deploy
```

Zie `CLAUDE.md` voor de werkwijze en de openstaande stappen (rebranding,
domeinkoppeling, CI-secrets).
